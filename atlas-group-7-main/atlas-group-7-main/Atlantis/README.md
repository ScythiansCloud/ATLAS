# Part 1: ATLAS event display

Welcome to part 1 of FP2 ATLAS Experiment laboratory. 

## Getting the setup ready
For the event display, you need to have java installed. If not, download the Java Development Kit (JDK) from https://www.oracle.com/java/technologies/downloads/#java11

## How-to start ATLANTIS
In your terminal, do:

```
java -jar atlantis.jar
```
